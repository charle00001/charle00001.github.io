第一种方法：#

这里有两个仓库：

https://github.com/xiazeyu/live2d-widget-models

https://github.com/xiazeyu/live2d-widget.js

就是https://cdn.jsdelivr.net/npm/live2d-widget@3.1.4/里面有一个文件夹，lib，这玩意里面有最重要的文件.

我之前（很以前很以前，年代久远）看到有人博客上写了，我就那样干了：
但是现在这玩意是很久以前的版本了，连上面两个GitHub仓库都没这代码，我只能找了一个他们的CDN服务器证明这些存在过.

（就是上面那个CDN）

大概是这样搞的：

  1. 把根目录/themes/PaperMod/layouts/partials/footer.html复制出一份到根目录/layouts/partials这个文件夹里面（没有这个文件夹的就新建一个），用随便什么代码编辑器打开（vi也可以）.

  2. 在<footer>和</footer>标签里面添加一段，用于管理models：

    <!-- Live2D，网页上的小人，模型都在里面 -->
    <!-- 你也可以把js文件下载下来，放到static/js/目录（只要是在本地博客文件夹里都行）下，就不依赖别人的服务了 -->
    <script type="text/javascript" src="https://cdn.jsdelivr.net/npm/live2d-widget@3.1.4/lib/L2Dwidget.0.min.js"></script>
    <script type="text/javascript" src="https://cdn.jsdelivr.net/npm/live2d-widget@3.1.4/lib/L2Dwidget.min.js"></script>

    <script type="text/javascript">
        L2Dwidget.init({
            model: {
                scale: 1,
                hHeadPos: 0.5,
                vHeadPos: 0.618,
                jsonPath: '/code/js/wanko/wanko.model.json',       // xxx.model.json 的路径,换人物修改这个
            },
            display: {
                superSample: 1,     // 超采样等级
                width: 240,         // canvas的宽度 120
                height: 600,        // canvas的高度 300
                position: 'left',   // 显示位置：左或右
                hOffset: 0,         // canvas水平偏移
                vOffset: 0,         // canvas垂直偏移
            },
            mobile: {
                show: false,         // 是否在移动设备上显示
                scale: 1,           // 移动设备上的缩放
                motion: true,       // 移动设备是否开启重力感应
            },
            react: {
                opacityDefault: 1,  // 默认透明度
                opacityOnHover: 1,  // 鼠标移上透明度
            },
        });
    </script>
