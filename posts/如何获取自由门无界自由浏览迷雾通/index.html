获取自由门、无界、自由浏览、迷雾通#

上一篇文章我们讲了如何获取Tor和I2P，但是这两个工具的速度太慢，如果仅仅是浏览youtube、tiktok、reddit这种【非敏感但封锁】的网站，【不需要】Tor和I2P.

现在我们有了一些【比它们俩个】更快的翻墙工具——自由门、无界、自由浏览、迷雾通，我将会和大家仔细说说怎么获取.

如果自己没有Tor或者别的破网软件不能下载，也可以去【顶栏—->别的&工具—->破网软件获取中心】来从onedrive/GitHub获取.#

自由门#

自由门是“全球网络自由联盟”研发的破网工具，缺点是【不】开源. 不过有很多人使用，应该没有后门.

官方下载地址：在这里

除了从官网和我的破网软件获取中心下载以外，自由门也可以这么获取：

发一封【主题不为空】的邮件到ziyoumen99@gmail.com（不要用国内邮箱），十分钟内会收到回信，回信中包含下载地址.

无界#

无界也是一个翻墙软件，官网在"这里"

无界浏览是美国无界网络公司建立在GIFT技术平台上的客户端软件，提供中国互联网用户绕过网络长城的方法。

无界的另一种获取方法很像Tor浏览器的"GetTor"服务，甚至比GetTor的选项还【多】，我们就暂且称它为"Get无界".

如何使用"Get无界"呢？

电子邮件获取

给xiazai@upup.info发一封电子邮件，切忌使用【国内】的电邮服务.

Skype

使用正版Skype搜索wujieliulan.com这个账号，打开它就可以获取无界下载网址. 另外给它发送任何消息也【可以】获取无界下载地址.

视窗即时传讯

用微软视窗即时传讯Windows Live Messenger给以下其中一个帐号发一个短信，就会收到几个有效IP地址：
wj2007001@hotmail.com， wj2007002@hotmail.com，wj2007003@hotmail.com， wj2007004@hotmail.com，wj2007005@hotmail.com， wj2007006@hotmail.com

Google Talk
Yahoo Messenger

Windows Live Messenger/Google Talk/Yahoo Messenger的服务已关闭，此（些）项获取方法作废.

迷雾通是一款先进的VPN，提供免费和付费. 免费限速125KB每秒，比Tor和I2P快.
